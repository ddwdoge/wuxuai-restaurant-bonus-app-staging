# 03_UX_REGELN

## Ziel

Dieses Dokument definiert die verbindlichen UX-Regeln für WUXUAI Bonus.

## Grundprinzipien

### Eine Seite = Eine Entscheidung

Jeder Bildschirm verfolgt genau ein Ziel.

### Mobile First

Zuerst für Smartphones entwerfen, danach Tablet und Desktop.

### Installationsassistent

Das Onboarding fühlt sich wie ein Einrichtungsassistent an, nicht wie
ein Formular.

### Autosave

Jede Änderung wird automatisch gespeichert. Kein manueller
Speichern-Button.

### Wenig Text

Icons und klare Begriffe statt langer Erklärungen.

### Keine technischen Begriffe

Restaurantbesitzer sollen keine Entwicklerbegriffe sehen.

### Dashboard

Dashboard beantwortet: - Wie läuft mein Bonusprogramm? - Was ist heute
passiert? - Was soll ich als Nächstes tun?

### Starter Kit

Im Onboarding gibt es nur einen Hauptdownload: Restaurant Starter Kit.

### Konsistenz

Alle Karten: - gleiche Höhe - gleiche Abstände - responsive - keine
abgeschnittenen Texte

### Deutsch

Alle sichtbaren Texte in V1 sind Deutsch.
